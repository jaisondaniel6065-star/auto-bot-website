# AI Engineer Academy — Website Launch Checklist

> Before you deploy, every box must be checked.

---

## 1. Content & Copy

- [ ] All pages have real content (no lorem ipsum or placeholder text)
- [ ] Spelling and grammar checked on every page
- [ ] Contact information is correct (email, phone, address)
- [ ] All links work (no broken links — test every single one)
- [ ] Call-to-action (CTA) buttons on every page
- [ ] Privacy policy page added
- [ ] Terms of service page added (if applicable)
- [ ] 404 error page designed (not the default ugly one)

---

## 2. Images & Media

- [ ] All images are compressed (use TinyPNG or Squoosh — under 200KB each)
- [ ] Images use proper formats (WebP for photos, SVG for icons/logos)
- [ ] Every image has alt text (for accessibility and SEO)
- [ ] No stretched or pixelated images
- [ ] Favicon uploaded (the small icon in the browser tab)
- [ ] Open Graph image set (the image that shows when shared on social media)
- [ ] Logo is crisp and properly sized

---

## 3. SEO (Search Engine Optimization)

- [ ] Unique page title for every page (under 60 characters)
- [ ] Meta description for every page (under 160 characters)
- [ ] Open Graph tags set (og:title, og:description, og:image, og:url)
- [ ] Twitter card tags set (twitter:card, twitter:title, twitter:description, twitter:image)
- [ ] H1 heading on every page (only one H1 per page)
- [ ] Heading hierarchy is correct (H1 → H2 → H3, not random)
- [ ] URL slugs are clean and readable (/about not /page?id=123)
- [ ] Sitemap.xml generated and submitted
- [ ] Robots.txt file configured
- [ ] Canonical URLs set to avoid duplicate content

---

## 4. Performance & Speed

- [ ] Page loads in under 3 seconds (test with Google PageSpeed Insights)
- [ ] Images are lazy loaded (load only when user scrolls to them)
- [ ] CSS and JavaScript files are minified
- [ ] No unused CSS or JavaScript loaded
- [ ] Fonts are optimized (use font-display: swap)
- [ ] Google PageSpeed score above 80 for both mobile and desktop

---

## 5. Mobile & Responsive

- [ ] Website looks perfect on mobile phones
- [ ] Website looks perfect on tablets
- [ ] Website looks perfect on desktop
- [ ] Text is readable without zooming on mobile
- [ ] Buttons and links are easy to tap on mobile (min 44×44px)
- [ ] Navigation menu works on mobile (hamburger menu)
- [ ] No horizontal scrolling on any device
- [ ] Tested on at least 3 different screen sizes

---

## 6. Security

- [ ] SSL certificate active (HTTPS, not HTTP — the padlock icon)
- [ ] Environment variables used for API keys (never hardcoded)
- [ ] No sensitive data exposed in frontend code
- [ ] Form inputs are validated and sanitized
- [ ] Rate limiting on forms to prevent spam
- [ ] CORS configured properly if using APIs

---

## 7. Forms & Functionality

- [ ] All forms submit correctly (test every single form)
- [ ] Form validation works (required fields, email format, etc.)
- [ ] Success/error messages show after form submission
- [ ] Email notifications arrive when forms are submitted
- [ ] Forms are protected against spam (honeypot or reCAPTCHA)
- [ ] All buttons do what they're supposed to do

---

## 8. Analytics & Tracking

- [ ] Google Analytics 4 installed
- [ ] Google Search Console connected
- [ ] Event tracking on key CTAs (button clicks, form submissions)
- [ ] Cookie consent banner added (required in many regions)

---

## 9. Accessibility

- [ ] All images have alt text
- [ ] Color contrast is sufficient (text readable against backgrounds)
- [ ] Website is navigable with keyboard only
- [ ] Focus states visible on interactive elements
- [ ] ARIA labels on interactive elements where needed
- [ ] Font size is at least 16px for body text

---

## 10. Pre-Launch Final Checks

- [ ] Custom domain connected and working
- [ ] WWW and non-WWW both redirect to the same URL
- [ ] Website works on Chrome, Firefox, Safari, and Edge
- [ ] All social media links point to correct profiles
- [ ] Website loads correctly when shared on WhatsApp, LinkedIn, Twitter
- [ ] Google PageSpeed Insights run — screenshot your score
- [ ] Backup of all code pushed to GitHub

---

## 11. Post-Launch

- [ ] Submit sitemap to Google Search Console
- [ ] Share the live link in the AI Engineer Academy community
- [ ] Add the live URL to your Project Tracker spreadsheet
- [ ] Take a screenshot and add it to your portfolio
- [ ] Celebrate — you just shipped a real project 🎊
